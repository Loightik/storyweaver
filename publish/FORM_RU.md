# Spigot - как заполнять форму (StoryWeaver)

Ссылка: https://www.spigotmc.org/resources/add
Сейчас выкладываем бесплатно (`FREE`).
**Категория Spigot:** `Misc`

## По порядку

1. **Title** - `StoryWeaver | ветвящиеся диалоги NPC`
2. **Category** - выбери **`Misc`**
3. **Version String** - `1.1`
4. **Tag Line** - из `tagline_ru.txt` (теги через запятую)
5. **Resource file** - Uploaded file → `../target/storyweaver-1.1.jar`
6. **Native Major MC Version** - `26.2`
7. **Tested Major MC Versions** - `1.16`, `1.17`, `1.18`, `1.19`, `1.20`, `1.21`, `26.2`
8. **Source Code** - URL GitHub (см. `GITHUB_RU.md`)
9. **Donation** - по желанию
10. **Languages** - `English, Russian`
11. **Description** - вставь `DESCRIPTION_RU.bbcode`
12. **Documentation** - `DOCUMENTATION_RU.bbcode`
13. **Additional Information URL** - репозиторий или README
14. **Alternative Support URL** - Issues / Discord
15. После сохранения - иконка 96×96 → `icon.png` (рядом `icon.jpg`)

Автор: **Loightik** (уже в `plugin.yml`).

Сначала `OVERVIEW_RU.md`. В текстах нет BBCode; подсказки `(жирный)` / `(наклонный)`.
Команды: `/storyweaver admin`, не `/storyweaveradmin`.
